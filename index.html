MovieRecommendation4000

Generates movie recommendations based on a user’s favourite films by comparing genre, duration, ratings and year of release. It uses a 2024 IMDb dataset of over 4,000 titles from Kaggle, preprocesses and normalizes the data, and employs cosine similarity with fuzzy matching; a simple Tkinter GUI lets users get recommendations.

SP500AnalyzedtoX

Automatically analyses all S&P 500 stocks (as of Aug 2024) by collecting ten‑year price data, financial statements and macroeconomic indicators via yFinance and FredAPI. It computes metrics such as average price, returns, beta and R² scores, performs dividend and ratio analyses and exposes the results through CSV files and a simple Flask API.

Regression‑Analysis‑Comparison

Compares regression models—linear, lasso, XGBoost, ARIMA and random forest—on S&P 500 stock prices using macroeconomic indicators like unemployment rate and federal funds rate. The 2012‑2023 monthly dataset is used to evaluate model performance (MSE and R²); linear regression emerged as the most effective.

PneumoniaDectection

A chest X‑ray classification project that uses transfer learning with a pretrained ResNet‑50 model in PyTorch to distinguish between normal and pneumonia scans. Data is split into training, validation and test sets, and the best‑performing model is saved.

BrainTumorMRIClassification

Image classification CNN built with PyTorch to categorize MRI scans into glioma, meningioma, pituitary tumour or normal. Uses a Kaggle dataset (~3,500 images), preprocesses and splits the data, trains for several hundred epochs and reports ~91% test accuracy.

LinearRegressionSentimentAnalysis

A learning project combining Microsoft stock opening prices with sentiment analysis of news headlines (using NLTK) from Feb 2022–Dec 2023. The data is gathered via Alpha Vantage API, sentiment scores computed and combined with stock data; a linear regression model predicts closing prices, achieving an R² around 0.84 despite high MSE.

Spotify2023DataVisualization

Visualizes the most streamed Spotify songs of 2023 in a Jupyter Notebook using pandas, Matplotlib and Seaborn. The notebook analyses song attributes and popularity across months and charts using data from Kaggle’s Top Spotify Songs 2023 dataset.
